About this Website

CMU Course List can provide you with more detailed information about courses at Carnegie Mellon University. From basic course information, such as course names, units, and department from SIO, to "student-oriented" information, such as scores in Rate-my-professor, average weekly workload from FCE semester surveys, and students comments under the course pages, you can obtain a wider horizon on courses at CMU.

You're encouraged to explore the website in any way you like! CMU Course List is designed like a search engine which allows users to search for courses based on course ID, professor, or any keyword. Moreover, if you're curious about building up professional skills from CMU curriculums, you can use the tags of skills below the search bar, which offers course recommendations for your own learning path.

The way CMU Course List processes log-in is simply a redirection to the CMU authentification, so your personal information will never be accssed by any third-party users. For comment boards on the course-page, it's viable for users to give comments anonymously.
